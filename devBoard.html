Question-1: What are the different ways to select an element in the DOM?

There are several way to select an element in the DOM. They are:

  • Selecting with ID :---document.getElementByID(ID).
  • Selecting with Class :---document.getElementByClass(Class).
  • Selecting with Tag :---document.getElementByTagName(Tag).
  • Selecting with Query Selector :---document.querySelector("#ID/.class/Tag").
  • Selecting with Query SelectorAll :---document.querySelector("#ID/.class/Tag").

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent are given below:

  • innerHTML : Inner HTML is the html inner html code in a element.
  • innerText : Inner Text is the text content of the element.
  • textContent : Text Content is also the text content of the element. But It returns textual content even for hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is the process of handling event , where always use event handler in the parent of target event. As a result, there is no need to use of event handler repeatedly.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process of event management, which is base on a concept that where an event occurs on a child element that bubble up ti its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

How I can create, add, and remove elements using JavaScript is given below :

  • To create Element : ---document.createElement("tagName").
  • To add Element : ---document.append(element) || ---document.appendChild(element)
  • To remove Element : ---document.remove(element) || ---document.removeChild(element)
